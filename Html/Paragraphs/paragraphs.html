Vamos ver nessa página, tipos tags que modificam fontes.

Primeiro veremos a função BOLD (Negrito) que usa o formato ">b<".
Outra forma de fazer o BOLD é usando a função STRONG dando força as letras. Possuindo o formator ">strong<"
Agora a função ITALIC (Itálico) que usa o formato ">i<".
Outra forma de fazer o italico é usando a função EMPHASIS dando ênfase à palavra. Possuindo o formato ">em<".
Essa é a função UNDERLINED (Sublinhado) que usa o formato ">u<".
Já essa é a função SUPERSCRIPTED (sobrescrito) que usa o formato ">sup<".
Essa é a função SUBSCRIPTED (subescrito) que usa o formato ">sub<".
Essa é uma função obsoleta. BIG é usada para aumentar o tamanho fonte sem o uso de CSS, seu formato é ">big<".
Essa é uma função obsoleta assim como big. SMALL é usada para diminuir o tamanho fonte sem o uso de CSS, seu formato é ">small<".

Outra forma de alterar as fontes é utilisando a sub-função STYLE. Dentro da identação do parágrafo. Como por exemplo:
">p style="font-family:'Times' "<"

Sendo possível declarar o possível uso de duas fontes distintas para o mesmo texto. Como por exeplo:
">p style="font-family: 'Courier', 'Arial' "<"
Mas nesse caso não sei como ele faz para escolher qual delas deve usar, e quando deve usar.

Além disso é possível fazer comentário nos códigos, comentários esses que não aparecem na página, somente no código fonte.
Basta utilizar o formato:
">!-- Comentário invisivel --<"

Lembrando que toda vez que os símbolos de Maior (>) e menor (<) aparecerem entre aspas ("") quer dizer que para utilizar deve invertê-los.
Não foram colocados corretamente para não confundir o interpretador do código, para que ele não ache que são tags
